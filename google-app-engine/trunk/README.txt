This package is a Google App Engine/Python based web site. It gets a list of topics
from <SOMEWHERE> and proceeds to create photo galleries for some of the topics based 
on an image search against Flickr. 

Technologies/Frameworks used:
* Google App Engine
* Python
* Django templates
* Jquery/JqueryUI
* Gaeflickrlib - A library for accessing flickr from within GAE (modified for this project where necessary)
